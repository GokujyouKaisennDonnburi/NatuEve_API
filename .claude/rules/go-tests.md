# Go テストルール

> 適用対象: `**/*_test.go`。
> このファイルは `.claude/rules/` から常時自動ロードされる（テストの新規作成時にも確実に適用するため、意図的にパススコープを付けていない）。

## 規約
- テーブル駆動テスト（table-driven tests）を基本とする
- テストヘルパーは `t.Helper()` を呼ぶ
- テストデータは `testdata/` ディレクトリに配置
- DB を使うテストは `TestMain` でセットアップ/テアダウン
- テストファイルは対象ファイルと同じパッケージに配置（`_test` サフィックス付きパッケージも可）
